1993

The first version of AOL software spanned three minor revisions. Version 1.0 was initially available for Macintosh only, and each diskette was operating system specific. Most were released on 3.5" floppy diskettes, however there was some outliers in the 5.25" range. Macintosh used several techniques to encode more data on standard disks than the DOS/Windows equivalent. Some of the earliest 1.0 versions of AOL software seem to be either in 400k MFS or 800k & 1.4MB HFS, so subequently they require age appropriate systems with the correct type of drive to read.

Not much is known about the packaging of these 1.0 versions. I have seen a 1.1 windows version in its package, which was a paper sleeve with cellophane window. It is also clear how widely these were distributed. They may have only been available by request, through catalog, bundles with some select magazines, or with a select purchase of related equipment.

The only diskette known to be earlier than the standalone 1.0 version for Macintosh would be the version that shipped with the original AOL charter membership.
